Empleo abierto es una plataforma que le permite a las personas encontrar empleo dentro de instituciones de gobierno. El usuario puede ver los empleos disponibles en todas las diferentes instituciones, asi como filtrar resultados por tipo de empleo, institución o rango de salario.

Los datasets para la aplicación son sacados de las siguientes dos ligas.

Remuneración mensual de los puestos en el Gobierno Federal: http://datamx.io/dataset/remuneracion-mensual-de-los-puestos-en-el-gobierno-federal

Directorio de funcionarios públicos del Gobierno Federal: http://datamx.io/dataset/directorio-de-funcionarios-publicos-del-gobierno-federal
